# Free AI Code Review That Actually Fixes Code

[Gitar](https://gitar.ai) is free AI code review that doesn't just comment - it fixes your code and validates every change against CI.

## What makes Gitar different

- **Actually fixes code** - not just suggestions you have to implement
- **Full codebase context** - catches subtle bugs other tools miss
- **CI validation** - every fix is validated against your pipeline
- **Free forever** - unlimited repos, no credit card required

## Getting started

1. **Install** - [Add Gitar](https://go.gitar.ai/free-ai-review) to your repositories
2. **Review** - Gitar automatically reviews your pull requests
3. **Fix** - comment `@gitar-bot fix` to apply suggested changes
