# AI code review that fixes your code automatically

[Gitar](https://gitar.ai) is AI code review that doesn't just comment, it fixes your code and validates every change against CI.

## What makes Gitar different

- **Fixes code** - not just suggestions you have to implement
- **Full codebase context** - catches subtle bugs other tools miss
- **CI validation** - every fix is validated against your pipeline
- **Run Automations** - automate workflows between PRs and linear, jira, slack and more

## Getting started

1. **Install** - [Add Gitar](https://go.gitar.ai/install) to your repositories
2. **Review** - Gitar automatically reviews your pull requests
3. **Fix** - comment `@gitar-bot fix` to apply suggested changes
